🍸 Bartenders Group - Projeto Integrador
📌 Sobre o projeto

O Bartenders Group é um projeto integrador desenvolvido durante o curso de Programação Web. Ele representa um site real voltado para apresentação de um bartender, seus serviços e catálogo de drinks, além de permitir o contato direto com clientes de forma simples e organizada.

Este foi o meu primeiro projeto full stack, onde atuei tanto no desenvolvimento do front-end quanto no back-end, aplicando na prática os conhecimentos adquiridos durante o curso e aprendendo novas tecnologias ao longo do processo.

🎯 Objetivo

O objetivo do projeto é facilitar a comunicação entre clientes e o bartender, permitindo:

Visualização dos drinks disponíveis
Conhecimento sobre o profissional e seus serviços
Envio de solicitações de atendimento através de formulário
Melhor organização das demandas de eventos e festas
🔐 Funcionalidades
Página pública com apresentação do bartender
Catálogo de drinks sem necessidade de login
Sistema de autenticação (login)
Área restrita “Personality” após login
Formulário de solicitação de eventos com:
Nome do cliente
E-mail e telefone
Quantidade de pessoas
Tipo e quantidade de drinks desejados

Após o envio, as informações são encaminhadas diretamente para o e-mail do bartender, facilitando o atendimento e organização dos pedidos.

💻 Tecnologias utilizadas
HTML5
CSS3
JavaScript
PHP
XAMPP (Ambiente local)
